# MyClipboard 设计规格

## 概述

MyClipboard 是一个 macOS 菜单栏剪贴板管理工具，自动记录剪贴板历史（文字和图片），支持搜索、标签、Pin 和预览功能。使用 Electron 构建，未来支持跨平台。

## 核心功能

1. **自动监听** — 后台监控系统剪贴板，复制的内容自动存入历史
2. **历史列表** — 菜单栏弹出面板，列表展示历史条目，点击即复制
3. **去重处理** — 重复内容的条目不新增，更新时间戳并移到列表最上方
4. **Pin 功能** — 常用内容可固定在顶部，不受自动排序影响
5. **标签系统** — 自由输入标签，自动补全已有标签，支持按标签筛选
6. **内容预览** — 截断文本和图片缩略图提供预览弹窗，展示更多内容
7. **链接识别** — 文本中的 URL 自动识别，点击可直接打开浏览器
8. **搜索过滤** — 搜索框实时过滤文本内容匹配的条目

## 技术栈

- **运行时**: Electron
- **UI**: 原生 HTML/CSS/JS（不引入前端框架，保持轻量）
- **存储**: JSON 文件（元数据） + 本地文件系统（图片）
- **构建**: electron-builder

## 数据存储

### 存储路径

```
~/Library/Application Support/myclipboard/
├── history.json      # 条目元数据
├── images/           # 图片文件，以 entry id 命名
└── settings.json     # 用户偏好
```

### history.json 结构

```json
{
  "version": 1,
  "entries": [
    {
      "id": "uuid-v4",
      "type": "text",
      "content": "复制的文本内容",
      "tags": ["design", "reference"],
      "pinned": false,
      "createdAt": 1717142400000,
      "updatedAt": 1717142500000
    },
    {
      "id": "uuid-v4",
      "type": "image",
      "imagePath": "images/abc123.png",
      "tags": [],
      "pinned": true,
      "createdAt": 1717142400000,
      "updatedAt": 1717142500000
    }
  ]
}
```

### 去重逻辑

- **文本条目**: 比较 `content` 字段，完全相同视为重复
- **图片条目**: 比较图片文件的 MD5 哈希值

去重时仅更新 `updatedAt`，不修改 `createdAt`。

### 内容上限

- 默认最多保留 500 条历史（可在 settings 中配置）
- 超过上限时自动删除最旧的非 Pin 条目
- 图片文件同步清理

```
┌──────────────────────────────────────────┐

│  ┌──────────────┐  ┌──────────────────┐  │
│  │ Tray Manager │  │ Clipboard Monitor│  │
│  │ - 菜单栏图标  │  │ - 定时轮询剪贴板  │  │
│  │ - 面板显隐    │  │ - 去重判断        │  │
│  │ - 点击事件    │  │ - 增量写入        │  │
│  └──────┬───────┘  └────────┬─────────┘  │
│         │                   │            │
│  ┌──────┴───────────────────┴──────────┐ │
│  │           Storage Layer             │ │
│  │  - history.json 读写                │ │
│  │  - 图片文件管理                      │ │
│  │  - settings 管理                    │ │
│  └──────────────┬──────────────────────┘ │
│                 │ IPC (contextBridge)     │
├─────────────────┼────────────────────────┤
│        Renderer Process                  │
│                 │                        │
│  ┌──────────────┴──────────────────────┐ │
│  │           Panel UI                  │ │
│  │  - 搜索框 + 快捷标签 + 列表         │ │

│  └─────────────────────────────────────┘ │
└──────────────────────────────────────────┘
```

### IPC 接口

| 方向 | 方法 | 说明 |
|------|------|------|
| Renderer → Main | `get-entries` | 获取历史列表（支持搜索/标签过滤参数） |
| Renderer → Main | `copy-entry` | 复制条目到剪贴板 |
| Renderer → Main | `delete-entry` | 删除条目 |
| Renderer → Main | `update-entry-tags` | 更新条目标签 |
| Renderer → Main | `toggle-pin` | 切换 Pin 状态 |
| Renderer → Main | `get-all-tags` | 获取所有已有标签（用于自动补全） |
| Renderer → Main | `clear-all` | 清空全部历史 |
| Renderer → Main | `get-settings` | 获取设置 |
| Renderer → Main | `update-settings` | 更新设置 |
| Main → Renderer | `entries-updated` | 条目列表变化时推送 |

## UI 设计

### 面板布局

```
┌─────────────────────────────────────────┐
│  🔍 搜索内容...                          │  ← 搜索框 (固定顶部)
├─────────────────────────────────────────┤
│  📌 常用标签: [设计] [参考] [后端]       │  ← 快捷标签行 (有内容可选时显示)
├─────────────────────────────────────────┤
│  ┌─────────────────────────────────────┐ │
│  │ 📌 [设计] [参考]                     │ │  ← Pin 条目高亮
│  │ 这是一段被复制的文本内容，它可能超... │ │  ← 文本截断显示
│  │ 3 分钟前              👁️ 📋 🗑️  │ │
│  ├─────────────────────────────────────┤ │
│  │    [后端]                            │ │  ← 普通条目
│  │ 另一段文字内容...                    │ │
│  │ 1 小时前              👁️ 📋 🗑️  │ │
│  ├─────────────────────────────────────┤ │
│  │ 🖼️ screenshot_2024.png              │ │  ← 图片条目，小缩略图
│  │ 2 小时前              👁️ 📋 🗑️  │ │
│  └─────────────────────────────────────┘ │
├─────────────────────────────────────────┤
│              清空全部历史                │  ← 底部操作
└─────────────────────────────────────────┘
```

- **面板尺寸**: 宽约 360px，高度自适应（最大约 600px，超出可滚动）
- **条目文本截断**: 单行截断约 50 字符，超出显示省略号
- **图片缩略图**: 36x36px 小缩略图，显示文件名
- **排序规则**: Pin 条目在上按更新时间倒序，非 Pin 条目在下按更新时间倒序

### 预览弹窗

```
┌───────────────────────────────────────┐
│  预览                              ✕  │
├───────────────────────────────────────┤
│                                       │
│  [图片: 最大 500x400px 等比缩放]       │
│  [文本: 最多 20 行, 支持选中和滚动]    │
│                                       │
├───────────────────────────────────────┤
│  [标签编辑区]        📋 复制    🗑️   │
└───────────────────────────────────────┘
```

- 预览弹窗固定尺寸约 520x500px
- 图片按比例缩放适配窗口，保留宽高比
- 文本显示增多到 20 行左右，超出可滚动，文字支持选中
- URL 在预览中仍然是可点击链接

### 交互行为

| 操作 | 行为 |
|------|------|
| 点击条目主体 | 复制内容到系统剪贴板，更新时间，移到顶部，面板关闭 |
| 点击 📋 | 同上 |
| 点击 👁️ | 打开预览弹窗 |
| 点击 🗑️ | 删除该条目 |
| 点击 📌 | 切换 Pin 状态 |
| 点击标签区 | 弹出输入框，自由输入标签，逗号或回车确认 |
| 搜索框输入 | 200ms 防抖，实时过滤文本内容匹配的条目 |
| 点击快捷标签 | 按标签筛选，再次点击取消筛选 |
| 点击外部 | 面板关闭 |
| 点击链接文本 | 用默认浏览器打开 |

### 标签输入

- 点击条目标签区弹出小型输入框
- 输入时下拉显示已有标签自动补全建议
- 逗号或回车确认一个标签，退格键删除末尾标签
- 标签以小圆角徽章形式显示在条目上

## 排除功能（不做）

- 密码管理器应用复制内容排除（未来迭代）
- 跨设备同步
- 富文本 / HTML 内容保留（仅纯文本）
- 快捷键触发面板（当前仅点击菜单栏图标）
- 多语言支持（当前仅中文 UI）

## 面板窗口行为

- `alwaysOnTop: true`，失焦自动隐藏
- 窗口类型: `type: 'panel'` (macOS) / `type: 'toolbar'` (Windows/Linux)
- 无边框窗口，自定义圆角和阴影
- 面板出现在菜单栏图标下方
